Lungomare Theme | Documentation

Lungomare theme is a perfect solution for restaurant/cafe/bar website type. It uses ZURB Foundation 5 framework and NOUVEAU theme framework. It's clean and simple, easy to customize and extend functionality, mobile first and responsive layout. It's packed with 4 plugins for restaurant type of a website: Gallery, Events, Slider, Food menu.

01. Getting Started

a. Theme Installation

To use this theme you must have WordPress(3.9 or newer) already installed.
Follow these steps to install the theme in the admin panel:

  1. Go to 'Appearance > Themes' section.
  2. Click 'Add New' and select the 'Upload Theme' option.
  3. Choose the lungomare.zip file and press 'Install Now'.
  4. Once the theme is uploaded you need to activate it by clicking 'Activate' link.

b. Plugin Installation

Lungomare theme comes with 4 plugins that are located in /plugin folder: schat_menu.zip, schat_events.zip, schat_gallery.zip, schat_slider.zip.

For each plugin perform these steps:

  1. Go to Plugins > Add New > Upload Plugin.
  2. Choose the plugin file (.zip) and press 'Install Now'.
  3. If successful, click Activate Plugin to activate it, or Return to Plugin Installer for further actions.

c. Demo Content

Importing the XML file will add posts, slides, gallery photos, categories, pages, comments, events, food menu categories, food menu items and navigation menu content.

  1. Locate the sample content file in the theme package (demo content folder).
  2. Go to Tools > Import and click WordPress, then Install. WordPress will ask you to install an additional plugin for data import.
  3. After you activate the plugin, just select the file from your theme package and follow the instructions.

d. Setting Default Homepage

1. Create Homepage (skip, if you installed demo content)
To set up the Home page you need to create a new page, Go to Pages > Add New. You can give this page a title "Home" yet you do not need to include any contents for now. Select the “Full-width template” template from the Page Attributes section and click 'Publish'.
2. Create Blog page (skip, if you installed demo content)
Go to Pages > Add New. You can give this page a title "Blog" or "News". The content area of the page should be empty. Select the “Page with sidebar” template from the Page Attributes section and click 'Publish'.
3. Configure Reading Settings
Now, we've created 2 pages: home and blog. To configure reading settings you need to visit WordPress Admin > Settings > Reading and choose that front page displays as static page. Then select 'Home' as Front Page and 'Blog/News' as Posts Page.

e. Setting Main Menu

To set up the main menu go to Appearance > Menus, create a menu, add links and assign it to "Primary Menu" area.

If you installed the demo content, you have 2 menus already created for you. Edit them and assign 'Main menu' to "Primary Menu" area.

To display 'Footer Menu', go to 'Appearance' > 'Widgets' and add a 'Custom Menu' widget to 'Footer' with 'Footer Menu' selected in the widget options.

02. Theme Settings

To adjust theme settings go to Appearance > Theme Settings in WordPress Admin.

Here you can set:

  • Copyright text that is displayed in the bottom of the website.
  • Google Analytics tracking code.
  • Address and phone number that will be displayed in the top right corner of the website.
  • Social Media Links.

03. Widgets and Sidebars

Lungomare theme has 3 predefined sidebars.

Blog Sidebar - displayed on blog pages
Site Sidebar - displayed on static pages.
Footer Sidebar - displayed in footer area on every page of the website.
To add/edit widgets follow this guide http://codex.wordpress.org/WordPress_Widgets

04. Theme Customizations

a. Using Customizer

To quickly adjust the appearance of the theme go to Appearance > Customize.
Here you can upload logo (header image setting), change website background image, adjust website colors in real time.

b. WP Admin Editor

Go to Appearance > Editor. Here you can edit slyle.css file to add your own css.
Lungomare theme uses Foundation framework, to learn about default css classes please visit official Foundation documentation page: http://foundation.zurb.com/docs

Theme Customizer and WP Admin Editor are only good for minor theme customization. In case you need to do major changes to the layout, it's better to work with SASS files (all the SASS files included in this theme). You may need to do a little setup. You can find setup directions for various use-cases in Zurb Foundation’s official SASS documentation.

05. Slider

Make sure that 'schat_slider' plugin is installed and activated, if so click on 'Slider' menu item in WordPress Admin. The Slider is automatically displayed on Home page if you have at least one slide added.

To add a new slide:

  1. Click 'Slider' > 'Add new'
  2. Write a title which will be displayed as a heading
  3. Set featured image (recommended size is 1200x400px)
  4. Use the visual editor to include some text with a slide
  5. Use can also add a button to your slide by filling in two fields: Link (button URL) and Link text (button text).

06. Restaurant Menu

Make sure that 'schat_menu' plugin is installed and activated, if so you are able to see 'Menu' menu item in WordPress Admin.

  1. To set up the Menu page you need to create a new page, Go to 'Pages > Add New'. You can give this page a title "Menu" you can add description to your 'Menu' page by using the visual editor. Select the 'Menu page' template from the Page Attributes section and click 'Publish'.
  2. Click 'Menu' > 'Menu Categories' and add categories of products you have. You can adjust the order of categories by editing 'order' field of category parameters.
  3. When the categories are set, click 'Menu' > 'Add new' to add new product. Don't forget to choose a category when adding a new product in order for it to be displayed on Menu page.

07. Gallery

Make sure that 'schat_gallery' plugin is installed and activated, if so click on 'Gallery' menu item in WordPress Admin.

  1. To set up the Gallery page you need to create a new page, Go to 'Pages > Add New'. You can give this page a title 'Gallery'. Select the 'Menu page' template from the Page Attributes section and click 'Publish'.
  2. Click 'Gallery' > 'Add new'
  3. Write a title which will be displayed as a caption
  4. Set featured image
  5. Optional: choose a photo category

08. Events

Make sure that 'schat_events' plugin is installed and activated, if so click on 'Gallery' menu item in WordPress Admin.
To add a new event:

  1. Click 'Events' > 'Add new'.
  2. Write a title of the event.
  3. Set featured image.
  4. Use the visual editor to include some text with about an event.
  5. Optional: fill in 'time' and 'location' fields in any format you prefer.

09. Shortcodes

Responsive Grid

[grid][/grid]
This shortcode will create a container for you grid.
[grid][1-2]Content A[/1-2][1-2]Content B[/1-2][/grid]
This shortcode will create a two column grid.
[grid][1-3]Content A[/1-3][1-3]Content B[/1-3][1-3]Content C[/1-3][/grid]
This shortcode will create a three column grid.
[grid][1-4]Content A[/1-4][1-4]Content B[/1-4][1-4]Content C[/1-4][/grid]
This shortcode will create a four column grid.
Note: you can also use html when adding/editing pages of the website. Foundation responsive grid.

Other shortcodes

[button link="url"]Text[/button]
This shortcode will create a button with provided url and title.
[social]
This shortcode will display social media buttons if you set up URL's on 'Theme Settings' page.
[recent-gallery]
This shortcode will display 5 latest photos from the gallery plugin. It should be used only on 'Full-width template' static pages.
[recent-events count="3"]
This shortcode will display the most recent events. 'Count' attribute specifies number of elements to be displayed, by default it's 3.
